CPU design
===

* 8-bit architecture
* 16-bit addressing, can combine D and E into one address.
* Little-endian

###Registers
* A - Accumulator
* B
* C
* D - Low byte for addressing
* E - High byte for addressing
* F - Flags
* PC - 16-bit program counter
* SP - 16-bit stack pointer

###Flags
* Z - Zero
* C - Carry
